# Onboarding Online — Grupo Inove Prime
## Como publicar no Netlify (5 minutos)

### Arquivos da pasta
```
ONBOARDING-ONLINE/
├── index.html      ← Página principal (tudo aqui)
├── logo-branco.png ← Logo branca (usada no splash e header)
├── logo-cor.png    ← Logo colorida (usada no header após login)
└── INSTRUCOES-DEPLOY.md
```

---

### Passo a passo para publicar

1. Acesse [netlify.com](https://app.netlify.com) e faça login

2. No painel, clique em **"Add new site"** → **"Deploy manually"**

3. **Arraste a pasta `ONBOARDING-ONLINE`** inteira para a área de upload do Netlify

4. Aguarde alguns segundos → Netlify gera uma URL tipo:
   `https://nome-aleatório.netlify.app`

5. Para ter uma URL amigável, clique em **"Site settings"** → **"Change site name"**
   - Sugestão: `onboarding-grupoinove`
   - URL final: `https://onboarding-grupoinove.netlify.app`

---

### Para atualizar o conteúdo depois

1. Edite o `index.html` (procure o texto que quer mudar)
2. Volte ao Netlify → site → **"Deploys"** → **"Deploy manually"**
3. Arraste a pasta novamente → publicado em segundos

---

### Como usar com novos funcionários

Envie o link por WhatsApp ou e-mail no primeiro dia:

> *"Olá [nome]! Acesse o nosso onboarding online antes do seu primeiro dia:*
> *🔗 https://onboarding-grupoinove.netlify.app*
> *Lá você vai conhecer o Grupo, as empresas, os sistemas e o cronograma da semana."*

---

### O que o funcionário encontra

| Etapa | Conteúdo |
|---|---|
| 1 | Identificação (nome, cargo, empresa, data) |
| 2 | Conheça o Grupo Inove Prime |
| 3 | As 3 Empresas (GCJ, IVI, Just Help) |
| 4 | Sistemas utilizados (Bitrix24, Datajuri, e-mail, WhatsApp) |
| 5 | Cronograma interativo da semana (checklist marcável) |
| 6 | ESG — Ambiental, Social e Governança |
| 7 | LGPD — Proteção de Dados |
| 8 | Documentos (5 termos para confirmar leitura) |
| 9 | Conclusão + próximos passos |

**Tudo salvo automaticamente no navegador do funcionário.**
